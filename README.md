# Bank Account Simulation (Java OOP)

This is a simple Java console application that simulates basic bank operations using Object-Oriented Programming.

## Features
- Create a bank account
- Deposit money
- Withdraw money
- Check balance
- View transaction history

## Technologies Used
- Java
- VS Code
- Terminal
